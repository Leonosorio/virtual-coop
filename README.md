# VirtualCoop

Aplicación web desarrollada en Ruby on Rails para la gestión de clientes y pagos.

## Funcionalidades

- Registro de clientes
- Bloqueo de clientes
- Registro de pagos (crédito y débito)
- Edición de pagos
- Dashboard con métricas financieras
- Exportación de pagos a Excel
- Sistema de autenticación (login/logout)

## Tecnologías

- Ruby on Rails
- PostgreSQL
- Bootstrap 5
- Chart.js

## Autor

Proyecto desarrollado como práctica profesional para gestión financiera.
